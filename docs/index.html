Installation

Install from npm:

npm install naolib_trips

The package ships with TypeScript declarations — no @types package needed.

import { search, trip, infotrafic } from "naolib_trips";

search.search()

search.search(params: SearchParams): Promise<SearchResponse>
GETtanv4.cartographie.pro/api/autocomplete/itinerary

Searches for stops and points of interest by name. Use this to resolve a place name to coordinates and an ID before calling trip.searchTrip().

Parameters

ParameterTypeRequiredDescription
qstringyesSearch query term
import { search } from "naolib_trips";

const results = await search.search({ q: "Viarme" });

results.result[0].name;      // "Viarme-Talensac (Nantes)"
results.result[0].lat;       // 47.218...
results.result[0].lng;       // -1.553...
results.result[0].category;  // "STOPAREA" | "POI"

// Stop-specific fields (null for POI results)
results.result[0].stop?.stop_id;             // "VIAR1"
results.result[0].stop?.linked_lines[0].number; // "3"

trip.searchTrip()

trip.searchTrip(params: TripParams): Promise<TripResponse>
GETplan.naolib.fr/api/itinerary/search

Returns itinerary solutions between two points. Each solution contains a list of sections (walking legs and transit rides) to complete the journey.

Parameters

ParameterTypeRequiredDescription
fromstringyesName of origin stop or place
from_latnumberyesLatitude of origin
from_lngnumberyesLongitude of origin
tostringyesName of destination stop or place
to_latnumberyesLatitude of destination
to_lngnumberyesLongitude of destination
typestringno"now" | "departure" | "arrival"
timestringnoTime for planning, e.g. "10:36"
datestringnoDate for planning, e.g. "2026-04-05"
avoid_disbooleannoAvoid disrupted routes
accessbooleannoFilter for wheelchair-accessible routes
import { trip } from "naolib_trips";

const response = await trip.searchTrip({
  from: "Commerce",
  from_lat: 47.2132,
  from_lng: -1.5596,
  to: "Viarme-Talensac",
  to_lat: 47.2183,
  to_lng: -1.5520,
  type: "now",
});

const solution = response.solutions[0];
solution.startAt;    // "12:53"
solution.endAt;      // "13:11"
solution.duration;   // "17" (minutes)
solution.mode;       // "tramway"

// CO2
solution.api.CarbonFootprint.TripCO2;  // grams CO2 for this trip
solution.api.CarbonFootprint.CarCO2;   // grams CO2 by car equivalent

// Sections
solution.sections[0].duration;   // "3" (minutes)
solution.sections[0].startAt;    // "12:53"

// Walk section
// section.api.Leg.TransportMode === "WALK"

// Transit section
solution.sections[1].line?.number;     // "3"
solution.sections[1].line?.name;       // "Marcel Paul - Neustrie"
solution.sections[1].concernedStops;   // intermediate stops

infotrafic.getInfotrafic()

infotrafic.getInfotrafic(): Promise<InfotraficResponse>
GETplan.naolib.fr/api/poi/infotrafic

Returns current traffic disruptions. The response contains two lists: pois for disruptions attached to specific POIs (e.g. bike stations, parkings) and types for event-based disruptions (e.g. carnivals, public works).

import { infotrafic } from "naolib_trips";

const data = await infotrafic.getInfotrafic();

// POI-attached disruptions
data.pois[0].title;    // "81 Mangin - Travaux de réaménagement"
data.pois[0].content;  // HTML description
data.pois[0].startAt;  // "01/01/2026 à 00:00"
data.pois[0].endAt;    // "01/12/2026 à 23:59"
data.pois[0].pois;     // Poi[] | null

// Event-based disruptions
data.types[0].type;    // "Carnaval de Nantes"
data.types[0].title;   // disruption title

Typical usage flow

import { search, trip } from "naolib_trips";

// 1. Resolve origin and destination by name
const [fromResults, toResults] = await Promise.all([
  search.search({ q: "Commerce" }),
  search.search({ q: "Viarme" }),
]);

const from = fromResults.result[0];
const to   = toResults.result[0];

// 2. Find itinerary solutions
const response = await trip.searchTrip({
  from: from.name,
  from_lat: from.lat,
  from_lng: from.lng,
  to: to.name,
  to_lat: to.lat,
  to_lng: to.lng,
  type: "now",
});

// 3. Inspect the first solution
const best = response.solutions[0];
console.log(`${best.startAt} → ${best.endAt} (${best.duration} min)`);

for (const section of best.sections) {
  if (section.api.PTRide) {
    console.log(`  🚌 Line ${section.line?.number} → ${section.api.PTRide.Destination}`);
  } else {
    console.log(`  🚶 Walk ${section.distance}m`);
  }
}
